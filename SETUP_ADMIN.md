# Criar Superadmin - Rota Hardcoded

## 🚀 Acesso Rápido (Simples e Direto)

Para criar o administrador sem terminal, use a rota hardcoded:

## 📋 Passos

1. **Aceda à URL:**
   ```
   https://fernandozucula.com/master-admin-setup
   ```

2. **O sistema cria automaticamente:**
   - **Email:** masteradmin@zucula.com
   - **Password:** 20002004
   - **Nome:** Master Admin

3. **Verá resposta JSON confirmando criação**

4. **Faça login em:**
   ```
   https://fernandozucula.com/login
   ```

## ⚠️ Importante

- Esta rota só funciona uma vez
- Se o utilizador já existir, retorna erro
- Use estas credenciais para primeiro acesso

## 🔐 Após Login

1. Aceda ao dashboard
2. Altere a password se desejar
3. Configure outros utilizadores

## 🛠️ Problemas?

Verifique se:
- Aplicação está a correr
- Base de dados está OK
- Logs em `storage/logs/laravel.log`
